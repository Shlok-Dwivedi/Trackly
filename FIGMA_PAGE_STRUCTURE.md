Trackly — Figma Design Brief
Pages to Design (Base Only)

1. Login

Centered card, no sidebar
Logo + app name at top
Email + password fields
Google sign in button
Toggle between login / signup


2. Dashboard

Sidebar (desktop) / Bottom nav (mobile)
Mini calendar + Today's Events side by side
4 stat cards in a row
2 charts side by side (donut + bar)
Upcoming events grid below


3. Events List

Search bar + filter button at top
Filter pills row (status + category)
Grid of event cards (3 col desktop, 1 col mobile)


4. Event Detail

Event title + status badge + category tag
Description block
Date/time/location block
Attendees list
Photos grid
Action buttons (Edit, Delete, Join, Export)


5. Calendar

Top controls (My Events toggle + Month/Week/List view switch)
Full calendar grid taking up the rest of the screen
Color coded events


6. Profile

Top card (avatar, name, role badge, department, bio)
4 stat cards
My Events section
My Photos grid
Notification toggles


7. Users Page

Page title
Table with columns: User, Email, Role dropdown, Current role badge
One row per user


8. Reports

3 summary stat cards
Bar chart (events per month)
Donut chart (by status)
Pie chart (by category)
Staff activity table


Shared Components to Design Once

Sidebar — logo, nav links, user info at bottom
Bottom Nav — 5 tabs for mobile
Event Card — title, status badge, category tag, date, location
Status Badge — Planned (blue) / Ongoing (amber) / Completed (green)
Notification Bell — with unread count bubble
Empty State — icon + message (reused everywhere)
Loading Spinner — centered, reused everywhere
